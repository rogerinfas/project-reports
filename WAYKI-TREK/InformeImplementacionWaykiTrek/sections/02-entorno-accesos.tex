\section{Entorno de Despliegue y Accesos}

\subsection{Entorno del Servidor (Hosting)}
Para asegurar alta disponibilidad, actualizaciones automáticas y seguridad de los datos sin incurrir en mantenimientos de infraestructura propios, se ha optado por el alojamiento en la nube nativa de Odoo.

\begin{itemize}
    \item \textbf{Plataforma:} Odoo SaaS (Software as a Service)
    \item \textbf{Versión del Sistema:} Odoo versión \textbf{saas-19.2}
    \item \textbf{Plan Contratado:} Plan estándar/personalizado que incluye el acceso a la herramienta de desarrollo \textbf{Odoo Studio}.
    \item \textbf{Capacidad Nominal:} Licenciamiento configurado para \textbf{4 usuarios} concurrentes.
\end{itemize}
