\section{Módulo de Contactos (Directorio Digital)}

El módulo de Contactos funciona como el CRM básico de Textiles y Confecciones Atlas E.I.R.L., permitiendo centralizar toda la información de los clientes (clubes deportivos, colegios, clientes minoristas) y proveedores en una única base de datos segura y en la nube.

\subsection{Visualización de Contactos}

Al ingresar al módulo, visualizará el listado de contactos registrados. El sistema permite cambiar la vista entre tarjetas (Kanban) o una tabla (Lista) para facilitar la búsqueda.

\subsection{Registro de un Nuevo Contacto}

Para registrar un nuevo cliente o proveedor, siga estos pasos:
\begin{enumerate}
    \item Haga clic en el botón \textbf{Nuevo}.
    \item Seleccione si el contacto es un \textbf{Individuo} o una \textbf{Compañía}.
    \item Ingrese el Nombre completo o Razón Social.
    \item Complete la información de contacto (Teléfono, WhatsApp, Correo y Dirección, especialmente importante para envíos a provincias como Cusco o Puno).
    \item Guarde los cambios mediante el icono de la nube.
\end{enumerate}
